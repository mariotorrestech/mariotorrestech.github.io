The CISSP covers eight domains of information security:

1. Security and Risk Management
2. Asset Security
3. Security Architecture and Engineering
4. Communication and Network Security
5. Identity and Access Management
6. Security Assessment and Testing
7. Security Operations
8. Software Development Security

### Why CISSP

Five years of hands-on identity and access management work at Fidelity Investments, supporting 70,000 users across CyberArk, Azure AD/Entra, Broadcom Top Secret, BeyondTrust, and more, gave me a strong operational foundation.

The CISSP validated that experience against an industry-recognized security framework and broadened my understanding of security domains beyond day-to-day operations.
